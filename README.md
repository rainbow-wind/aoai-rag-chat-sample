# Azure RAG AOAI Chat Sample

## Project Overview

This project is to create a chat app based on RAG (Retrieval Augmented Generation) and vector search according to uploaded files like PDF. The application allows users to upload documents and interact with them through an intelligent chat interface powered by Azure OpenAI services.

## Features

- Document upload and processing (PDF and other file formats)
- Vector search capabilities for document content
- RAG-based chat interface
- Integration with Azure OpenAI services
- Modern web interface built with Next.js
